Project Overview

This was a personal project with the purpose to put in practice my UI skills. Applying
concepts such as contrast, hierarchy and maintaining a good visual design that gives
confidence to the user.

The challenge

To create a good looking user interface that also helps de user in finding the
main options of the app easily.

The process:

- Making an study of similar apps in the market.
- Drawing the sketches of the main screens of the app.
- Creating a style guide to maintain consistency among the project.
- Developing the mock ups of the app.
